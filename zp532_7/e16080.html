アンダーカバー 【 UNDERCOVER 】とは
1990年に 高橋盾 氏は ヴァンダライズ 【 VANDALIZE 】の 一之瀬弘法 氏と UNDERCOVER をスタート。
その後、1993年に アベイジングエイプ 【 A BATHING APE (略語 BAPE ベイプ エイプ ABATHINGAPE )】の NIGO 氏と 「 ノーウェア（ NOWHERE ）」を設立。
高橋盾氏の愛称「 JONIO ( ジョニオ )」は、文化服装学院在学中に セックス・ピストルズ のコピーバンド「東京 セックスピストルズ 」のボーカルを担当していた時のもので本家の ジョニー・ロットン にそっくりだった為、その愛称が付きました。
1994年に エーエフエフエー ( AFFA )を 藤原ヒロシ 氏と立ち上げる。
2022年2月に LAST ORGY 2（ ラストオージー 2 LASTORGY ）を復活させた。
デザイナー VERDY 氏の手掛ける Girls Don’t Cry ( ガールズドントクライ )、 KAWS 氏（ カウズ ) 、 ヴァージル・アブロー 氏の オフホワイト ( Off-White )、 HUMANMADE ( ヒューマンメイド )や ダブルタップス ( WTAPS )、 ネイバーフッド ( NEIGHBORHOOD )、 グッドイナフ ( GOODENOUGH )、 フラグメント ( FRAGMENT DESIGN フラグメントデザイン )、 シュプリーム ( Supreme )、吉田カバン ( PORTER 、 ポーター )、 バウンティハンター ( BOUNTY HUNTER ) 、 ステューシー ( STUSSY )、 ナイキ ( NIKE ) 、 ヴァンズ ( VANS )、 ドクターマーチン ( Dr.Martens )、 サカイ ( sacai )、 リーバイス ( LEVI’S )、 コムデギャルソン 、 エレクトリックコテージ 等と コラボ している。
